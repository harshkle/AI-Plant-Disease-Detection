# 🌿 AI Plant Disease Detection

## 📌 Description

This project is an AI-based system that detects plant diseases using Machine Learning techniques. It helps farmers and users identify diseases early and take proper actions to improve crop health.

---

## 🚀 Features

* Detects plant diseases from images
* Uses Machine Learning model
* Easy to use interface
* Fast and accurate predictions

---

## 🛠️ Technologies Used

* Python
* Machine Learning
* OpenCV (for image processing)
* NumPy / Pandas

---

## 📂 Project Structure

```
AI-Plant-Disease-Detection/
│── dataset/
│── model/
│── main.py
│── requirements.txt
│── README.md
```

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/harshkle/AI-Plant-Disease-Detection.git
```

2. Go to project folder:

```
cd AI-Plant-Disease-Detection
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the project:

```
python main.py
```

## 📊 Output
